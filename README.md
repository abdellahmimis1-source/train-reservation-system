# 🚆 Train Reservation System

Une application web complète de réservation de trains développée avec **Spring Boot** et **React**.

---

## 📌 Description

Cette application permet aux utilisateurs de rechercher des trajets, réserver des places et gérer leurs réservations. Elle dispose également d'un espace administrateur permettant la gestion des trains, des utilisateurs, des réservations et des statistiques.

---

## ✨ Fonctionnalités

- Développement d'une application complète de recherche de trajets, réservation de places et gestion des réservations.
- Création d'un espace administrateur pour gérer les trains, les utilisateurs, les réservations et les statistiques.
- Intégration de notifications par e-mail et WhatsApp.
- Mise en place d'une liste d'attente basée sur l'algorithme FIFO.
- Authentification sécurisée des utilisateurs.
- Tableau de bord administrateur avec statistiques.

---

## 🛠️ Technologies utilisées

### Backend
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Maven

### Frontend
- React
- Bootstrap
- Axios

### APIs
- Gmail SMTP
- Twilio WhatsApp API

## 🚀 Installation

### Backend

```bash
git clone https://github.com/abdellahmimis1-source/train-reservation-system.git
cd train-reservation-system
mvn spring-boot:run
```

### Frontend

```bash
cd train-frontend
npm install
npm start
```
